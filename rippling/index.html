  • Design end-to-end Search & AI experience — universal search to embedded assistants — fast, reliable, explainable, radically simple
  • Familiarity with LLMs, embeddings / vector search, RAG, tool / agent orchestration, evaluation / observability, safety / privacy
  • Design interplay of deterministic systems + probabilistic models; model failure states and recovery; transparent, user-controllable experiences
  • Build reusable capabilities — ranking / filtering, prompt governance, eval frameworks, UI components — that scale across product suite
  • Conversation design + interaction prototyping skills; partner with Product, Eng, Applied ML, GTM
  • Systems thinking; simplify complexity in compliance-heavy environments

Why I fit

  • Agentic AI interaction protocols, authored. WorkPal is exactly this shape — autonomous agent system with LLM-driven interfaces, intent inference over rigid UI toggles, human-in-the-loop moments, and Overview as agent observability. I've built the platform primitives this JD describes — not claiming familiarity, authoring them.
  • Deterministic + probabilistic, blended. WorkPal combines a deterministic project/task structure (rule-based routing, explicit permissions, verifiable state) with probabilistic agent layers (LLM-driven meeting summaries, intent inference, cross-platform retrieval). I've designed the handoff patterns — when to trust the model, when to surface sources, when to require user confirmation — and the graceful fallbacks when the probabilistic layer is uncertain.
  • Reusable primitives across products. WorkPal: agent observability (Overview), meeting-summary blocks, task-extraction UI, and confidence/source surfaces — plugged into multiple agent flows. Walmart Last Mile: shared workflow and UI patterns across driver, dispatch, and customer surfaces.
  • Complexity → radical simplicity. Walmart Last Mile: multi-actor workflow across driver, dispatch, customer, ops. Tencent SNS: cross-product unified experience across QQLink, QZone, Tencent Music. "Simplify without reducing power" is literally my track record.
